Download

There are several ways to download RSpec. Which way is right for you depends on how you want to use it.

Rubygems

This is the preferred way for non-Rails projects. Non-windows users may have to use sudo. Download the latest with:

gem install rspec

or if you want an older version:

gem install rspec -v 0.9.4

Ruby on Rails projects

See Rails

Direct download

Get it from Rubyforge

Subversion

Check it out:

svn checkout svn://rubyforge.org/var/svn/rspec/trunk rspec_trunk

or

svn checkout svn://rubyforge.org/var/svn/rspec/tags/REL_1_0_8/rspec rspec_REL_1_0_8

See the README for details about how to build it.
